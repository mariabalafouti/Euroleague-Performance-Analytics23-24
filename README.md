# Euroleague 2023-24: Business Analytics Project

This project was developed for the **Business Analytics** course at the **University of Macedonia (UoM)**.

## Academic Context
- **Course:** Business Analytics
- **Institution:** University of Macedonia, Thessaloniki, Greece.
- **Objective:** Apply statistical methods and data visualization using R to analyze sports performance data.

## Live Report
View the full interactive analysis here: [https://rpubs.com/mariabalafouti/1412951]

## Features
- **Data Analysis:** Descriptive statistics and correlation exploration (Assists vs. Turnovers vs. Points).
- **Visualization:** Custom-styled charts (Scatterplots, Boxplots, Histograms) using `ggplot2`.
- **Custom UI:** Enhanced HTML reporting with Montserrat fonts and glassmorphism styling.

## Files in this Repository
- `filename.Rmd`: The source code of the analysis.
- `filename.html`: The final rendered report.
- `basket2.jpeg`: Background image used for the report styling.
